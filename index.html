HTML

HTML (HyperText Markup Language) is the standard language used to create and structure web pages. It defines the structure and content of a website through "tags" or "markers," such as h1, p, a, among others, which indicate the type of content (titles, paragraphs, links, etc.) and how it should be displayed. HTML is essential for building any web page and serves as a foundation for integrating other technologies, such as CSS and JavaScript, that enhance the visual and functional aspects of the site.

CSS

CSS (Cascading Style Sheets) is a language that styles web pages, allowing control over the visual presentation separately from HTML. With CSS, it is possible to adjust colors, fonts, sizes, and spacing, as well as create responsive layouts that adapt to different devices. The language also offers features such as animations and transitions, and frameworks like Bootstrap facilitate development and ensure a consistent design.p>

JAVASCRIPT

JavaScript is a programming language used to add interactivity and dynamism to web pages. With it, you can create animations, validate forms, and manipulate page elements in response to user actions. In addition to being fundamental in front-end development, JavaScript is also used in the back-end with platforms like Node.js, making it essential for building modern web applications.

PYTHON

Python is a high-level programming language known for its simplicity and versatility. It is widely used in various fields, such as web development, data science, artificial intelligence, and automation. Python has a clear syntax and a vast library of modules, making it easier to create complex projects and allowing developers to focus on the program's logic.

C#

C# (C Sharp) is a programming language developed by Microsoft, designed to be simple and powerful. It is widely used to create desktop, web, and game applications, especially with the .NET platform. C# is an object-oriented language that supports concepts like encapsulation, inheritance, and polymorphism, allowing for organized and efficient development of complex systems.

TYPESCRIPT
(Studying)

TypeScript is a programming language developed by Microsoft that is based on JavaScript, adding static typing and object-oriented programming features. It allows developers to identify errors during development, improving code quality. TypeScript is widely used in large-scale projects and compiles to JavaScript, ensuring compatibility with any environment that supports this language.
